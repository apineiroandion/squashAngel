# Squash

## 1 Paso

El lider crea un proyecto, un repositorio,
 hace dos commits e invita al colaborador.

## 2 Paso

Paralelamente el lider y el colaborador hacen tres commits cada uno

## 3 Paso

El colaborador hace un push y el lider un fetch

## 4 Paso

El lider hace un squash de los commits del colaborado, y finalemente el de los suyos, quedando los commits de cada participante en sus respectivas ramas, yen la main solo los squash con todos
